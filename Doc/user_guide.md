# FindRegister - 사용자 및 조작 매뉴얼

FindRegister는 브라우저 상에서 인공지능/컴퓨터 비전을 활용해 저항을 손쉽게 식별하고 찾아낼 수 있도록 돕는 스마트 웹 애플리케이션입니다. 본 가이드는 본 앱의 주요 기능 사용법 및 저항 색띠 읽기 기초에 대해 기술합니다.

---

## 1. 저항 식별기 (Scanner) 사용법

저항 식별 기능은 PC의 웹캠 또는 모바일 기기의 카메라를 연동하거나, 직접 촬영한 저항 사진을 업로드하여 일련의 저항들을 자동으로 탐지하는 기능입니다.

### 1.1 실시간 카메라 모드
1.  **카메라 켜기**: 화면 중앙 하단의 `카메라 켜기` 버튼을 누르면 브라우저의 카메라 권한 요청 창이 뜹니다. `허용`을 선택합니다.
2.  **카메라 자동 감지**:
    *   **PC 환경**: 기본 장착된 전면 웹캠이 연동됩니다.
    *   **모바일 환경**: 초점 정밀도를 위해 **후면 카메라(Environment Camera)**가 자동으로 우선 구동됩니다.
3.  **카메라 전환**: 기기에 카메라가 여러 개일 경우 `카메라 전환` 버튼을 누르면 순차적으로 다른 렌즈로 화면이 순환 전환됩니다. (광각 렌즈나 접사 렌즈가 있는 경우 유용합니다).
4.  **스캔 수행**: 카메라 뷰포트 정중앙 가이드 점선 상자 부근에 저항을 수평 또는 비스듬하게 위치시키면 자동으로 저항 외곽에 **하늘색 테두리 상자(Bounding Box)**가 생성되며 계산된 옴(Ω) 값이 실시간으로 표시됩니다.
5.  **플래시(Torch) 켜기**: 모바일 기기 브라우저에서 플래시 제어를 지원하는 경우 `플래시` 버튼이 활성화되어 카메라 손전등을 켜고 끌 수 있습니다. (조명이 어두울 때 오차율을 크게 줄여줍니다).

### 1.2 사진 업로드 모드
1.  카메라 조작 버튼 우측의 `사진 업로드` 버튼을 눌러 이미 촬영된 스마트폰 갤러리의 사진이나 PC의 이미지 파일을 선택합니다.
2.  사진이 업로드되면 즉시 전체 스캔 알고리즘이 1회 실행되어 저항이 감지된 곳에 테두리 상자가 그려지고 우측 결과 목록에 등록됩니다.

### 1.3 감도 및 화질 튜닝
조명 상태에 따라 색띠 인식이 불안정할 때, 우측 하단의 **"화질 및 감도 튜닝"** 패널을 통해 캘리브레이션을 진행할 수 있습니다.
*   **밝기(Brightness)**: 주변 그림자가 너무 강해 갈색/검은색 띠가 뭉개질 때 높여주거나, 빛 반사가 심할 때 낮춥니다.
*   **대비(Contrast)**: 색띠의 경계가 희미할 때 대비를 올려주면 윤곽 추출이 명확해집니다.
*   **색포화도(Saturation)**: 띠의 색감(빨강, 주황, 파랑 등)이 흐릿하게 보일 때 색포화도를 120%~150% 수준으로 올리면 분류 확률이 높아집니다.
*   **감지 모드**: 특정 저항 배경색에 집중할 수 있습니다.
    *   *황토색 저항 전용* (일반 탄소 피막 저항)
    *   *파란색 저항 전용* (정밀 금속 피막 저항)
    *   *모두 감지* (기본 설정)

### 1.4 결과 상세 제어 및 수동 교정
검출된 사각형 박스 또는 우측 목록의 카드를 클릭하면 **"선택된 저항 상세 정보"** 패널이 열립니다.
1.  **읽는 방향 반전**: 저항이 180도 뒤집혀 인식되었을 경우 `읽는 방향 반전` 버튼을 누르면 띠의 순서가 역순으로 뒤집혀 다시 연산됩니다.
2.  **색띠 수동 보정**: 실내 조명 색온도로 인해 특정 색이 다르게 판독된 경우 (예: 빨간색이 주황색으로 오독됨), 색띠별 드롭다운을 선택해 올바른 색으로 즉시 변경할 수 있습니다.
3.  **생성기로 내보내기**: 검출된 값을 기반으로 색띠 이미지를 더 선명한 가상 그래픽으로 보고 싶거나 편집하고 싶을 때 누르면 `생성기` 탭으로 전환되며 값이 연동됩니다.

## 2. 저항 색띠 생성기 (Generator) 사용법

원하는 저항값을 지정하면 표준 색띠 배열과 실물에 가까운 벡터 그래픽(SVG)을 생성해 줍니다.

1.  **저항값 입력**: 입력란에 찾고자 하는 저항값을 자유롭게 적습니다.
    *   숫자만 입력 (예: `150` ➔ 150 Ω)
    *   접미사 입력 (예: `2.2k` ➔ 2.2 kΩ, `4M7` ➔ 4.7 MΩ, `0r22` ➔ 0.22 Ω)
2.  **색띠 모델 및 오차율 선택**:
    *   `4색띠` 또는 `5색띠` 토글 버튼을 선택합니다.
    *   오차율 드롭다운에서 오차 색띠를 지정합니다 (금색 5%, 갈색 1% 등).
3.  **그리기**: `그리기` 버튼이나 엔터 키를 치면 우측에 입체감이 살아있는 저항 SVG가 렌더링됩니다.
4.  **표준 저항 권장 기능**:
    *   만약 사용자가 비표준 저항값(예: 103옴)을 입력한 경우, 경고 메시지와 함께 가장 인접한 규격의 **E-계열 표준값(예: 91옴, 100옴, 110옴)** 칩이 표시됩니다. 칩을 클릭하면 바로 해당 표준값 저항이 생성됩니다.

## 3. 저항 색띠 읽는 법 기초

| 색상 | 수치 | 승수 (Multiplier) | 오차율 (Tolerance) |
|:---:|:---:|:---:|:---:|
| **검정 (Black)** | 0 | $10^0 = 1$ | - |
| **갈색 (Brown)** | 1 | $10^1 = 10$ | ±1% |
| **빨강 (Red)** | 2 | $10^2 = 100$ | ±2% |
| **주황 (Orange)** | 3 | $10^3 = 1\text{k}$ | ±0.05% |
| **노랑 (Yellow)** | 4 | $10^4 = 10\text{k}$ | ±0.02% |
| **초록 (Green)** | 5 | $10^5 = 100\text{k}$ | ±0.5% |
| **파랑 (Blue)** | 6 | $10^6 = 1\text{M}$ | ±0.25% |
| **보라 (Violet)** | 7 | $10^7 = 10\text{M}$ | ±0.1% |
| **회색 (Grey)** | 8 | $10^8 = 100\text{M}$ | ±0.01% |
| **흰색 (White)** | 9 | $10^9 = 1\text{G}$ | - |
| **금색 (Gold)** | - | $10^{-1} = 0.1$ | ±5% |
| **은색 (Silver)** | - | $10^{-2} = 0.01$ | ±10% |

### 4색띠 저항 (일반형)
*   **1번째 띠**: 첫 번째 자릿수 숫자
*   **2번째 띠**: 두 번째 자릿수 숫자
*   **3번째 띠**: 승수 (10의 거듭제곱 수, 앞의 두 자릿수에 곱해지는 값)
*   **4번째 띠**: 오차율 (보통 금색 ±5% 혹은 은색 ±10%)

> [!TIP]
> **예시 (100옴 저항)**
> *   갈색(1) - 검정(0) - 갈색(x10) - 금색(±5%)
> *   수식: $10 \times 10 = 100\ \Omega$ (오차범위 95Ω ~ 105Ω)

### 5색띠 저항 (정밀형)
*   **1~3번째 띠**: 각각 첫 번째, 두 번째, 세 번째 자릿수 숫자
*   **4번째 띠**: 승수 (곱해지는 값)
*   **5번째 띠**: 오차율 (보통 정밀용 갈색 ±1%)

> [!TIP]
> **예시 (2.2k옴 저항)**
> *   빨강(2) - 빨강(2) - 검정(0) - 갈색(x10) - 갈색(±1%)
> *   수식: $220 \times 10 = 2200\ \Omega = 2.2\ \text{k}\Omega$
